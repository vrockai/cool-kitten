Research

Beast SVN access

Beast sources can be accessed by svn with beast/beast account. The sources can be obtained from http://beast.unfuddle.com/svn/beast_beast/ with the command:

svn export http://beast.unfuddle.com/svn/beast_beast/ beast_svn

Enter beast as username and password when asked for one. Beast can be then run with the command:

ant run-gui

In case of any problems/comments/feature requests please feel free to e-mail me on viliam.rockai(at)gmail.com.


(VERY OLD) Beast - software used in experiments


Actual version of beast is available in svn repository. Please use the version below only if you have problems with running the SVN version. Jar archives below can be very old and missing some important functionality. Beast - source files
Beast - jar binary + used libraries

Usage:

java -Xmx1024m -jar Beast.jar ( or -Xmx2048, depends on korpus size )

Learning ( type within the interface ):

-buildlex path_to_corpus/subpath_to_corpus
-learn path_to_corpus/subpath_to_corpus

-help

Useful commands:

-treshold value : set the treshold - it is recomended to start with treshold of value about 4
-find token : search in lexicon for token
-synonyms token : computes synonyms for token
-relate token1,token2 : compute relatedness of token pair
